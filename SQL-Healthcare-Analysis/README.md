# Healthcare Data Analysis Using SQL

## Overview
This project demonstrates how SQL can be used to analyze healthcare utilization data, summarize patient visit patterns, identify cost drivers, and evaluate trends across facilities, counties, age groups, and diagnosis categories.

## Tools Used
- SQL
- PostgreSQL-style queries

## Analysis Performed
- Counted total visits and unique patients
- Summarized cost by diagnosis category
- Evaluated average length of stay
- Analyzed utilization by gender and age group
- Identified top facilities by total cost
- Produced county-level summaries
- Created monthly visit trends
- Flagged patients with multiple visits
- Performed basic data quality checks

## Key Skills Demonstrated
- SELECT statements
- GROUP BY
- CASE WHEN
- Aggregations
- Trend analysis
- Data quality validation

## Business / Healthcare Value
These queries can help healthcare organizations monitor utilization, identify high-cost service areas, track population patterns, and improve operational reporting.
